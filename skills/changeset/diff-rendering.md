# Diff Rendering

## v1 Purpose

Show reviewable file/hunk diffs for AutoDSM workspace changes before local PR creation or publish.

## Required Actions

- approve hunk
- reject hunk
- discard hunk/all with confirmation
- commit locally
- create local PR record

## Boundary

Diffs are for files under the active AutoDSM workspace. Production repo diffs are future/import scope.
