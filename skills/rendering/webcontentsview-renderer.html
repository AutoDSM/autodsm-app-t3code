Preview WebContentsView — Topology & Message Flow

Sibling child view under a BaseWindow. Not an iframe. Owned by PreviewViewController.
Blue dashed = typed RPC into controller. Green = controller → preview webContents.send / capture. Orange = preview HTTP/WS to sidecar (loopback only).
Channel Direction Schema Purpose
preview:set-bounds UI → main SetBoundsRequest Position/size preview view
preview:load UI → main RenderPlan Load a component at a given state
preview:state main → preview PreviewPropsAndTheme Live props/theme without reload
preview:capture UI → main → preview CaptureOptions / ScreenshotId CDP screenshot pipeline
preview:input UI → main → preview InputEvent Dispatch synthetic input via CDP
preview:diagnostics preview → main → UI DiagnosticsEvent Console, errors, denials, timings
